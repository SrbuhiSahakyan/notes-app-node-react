# notes-app-node-react

## Описание
Это приложение заметок, где пользователь может:
- Создавать, редактировать, удалять и просматривать заметки (CRUD)
- Устанавливать приоритет заметки: Низкий, Средний, Высокий
- Регистрация и вход с JWT: каждая заметка привязана к пользователю

---

## Функционал
- Backend: Node.js + Express + Prisma + PostgreSQL
- Frontend: React
- CRUD заметок
- JWT аутентификация
- Стилизация компонентов (inline-стили)
- Docker для быстрого развертывания

---

## API

### Auth
| Метод | Путь | Описание |
|-------|------|----------|
| POST  | `/auth/register` | Регистрация пользователя |
| POST  | `/auth/login` | Вход пользователя, возвращает JWT |
| GET   | `/auth/me` | Данные текущего пользователя |

### Notes (требуется JWT)
| Метод | Путь | Описание |
|-------|------|----------|
| GET    | `/notes`      | Все заметки пользователя |
| GET    | `/notes/high` | Только заметки с высоким приоритетом |
| POST   | `/notes`      | Создать заметку |
| PUT    | `/notes/:id`  | Обновить заметку по ID |
| DELETE | `/notes/:id`  | Удалить заметку по ID |

## Запуск проекта через Docker (пошагово)

**Клонируйте репозиторий**

```bash
git clone <URL_репозитория>
cd notes-app-node-react

**Создайте файл .env для backend**

cp backend/.env.example backend/.env

Соберите Docker-образы без кэша

docker-compose build --no-cache

Поднимите контейнеры

docker-compose up -d

Проверьте, что все контейнеры запущены

docker-compose ps

docker compose exec backend npx prisma migrate dev --name init

Откройте проект в браузере

Узнайте IP вашего компьютера в сети (например, 192.168.0.109)

Фронтенд: http://<ваш_IP_компьютера>:3000

Backend API: http://<ваш_IP_компьютера>:4000
